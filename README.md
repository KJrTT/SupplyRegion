### SupplyRegion — Учёт заявок на закупку

Настольное приложение учета заявок снабжения для компании "Регионснаб". 
Приложение позволяет вести единый реестр заявок, контролировать статусы и быстро выгружать данные в CSV.


### Возможности
- Создание заявки в отдельном окне
- Изменение статуса заявки
- Фильтр по статусу
- Поиск по наименованию товара / отделу / инициатору
- Экспорт в csv
- Подсчет средней цены
- Процент завершения
- Список последних действий
- Подсчет общей суммы
- Счетчик заявок


### Запуск приложения
Для запуска приложения достаточно скачать zip - архив и разархивировать его. После этого ищем исполняющий файл (.exe) и 2 раза кликаем.

[Ссылка на скачивание](https://github.com/KJrTT/SupplyRegion/releases/tag/Exe-file)
